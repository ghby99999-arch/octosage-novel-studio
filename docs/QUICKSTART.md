# Quick Start

This is the shortest path for a non-developer.

## 1. Install and open

```powershell
npm.cmd install
npm.cmd run desktop
```

## 2. Start a book

On the home page:

1. Type one sentence about the story idea.
2. Click `开始`.
3. Pick a template if you want a stronger starting direction.

## 3. Keep writing

- Use `写 · 创作中心` to continue the current chapter.
- Use the batch buttons only when you want the app to keep going for several chapters.
- Use `统 · 孵化数据` to see progress, grades, cost, and recent activity.

## 4. Export when needed

- Use `影 · 视频工厂` to export screenplay, storyboard, and video prompts.
- Use `发 · 投稿发布` to prepare the publish package and browser-assisted publish flow.

## 5. Ask for help

Open `设 · 系统配置`, then use:

- `快速开始`
- `使用手册`
- `更新记录`
- `导出诊断包`
- `联系客服微信`

The diagnostic file is local and is meant to help support reproduce the problem without exposing raw source prose.
